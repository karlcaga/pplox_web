Complete expression parser

With the release of pplox 0.0.4 we now have complete support for parsing expressions. These expressions are parsed into a parse tree for evaluation, but currently expressions are not evaluated.

We support math expressions:

1 + 2 - 3
1 * 2 / 3

Comparisons between numbers are also supported:

1 < 2 <= 3
3 > 2 >= 1

And finally, equality checks between strings are now also parsed:

"foo" == "foo"
"foo" != "bar"

Next up, we will be working on evaluating these expressions.

Full tokenizer and Dockerfile

Initial release of pplox_web only supported single character tokens. Updated to version 0.0.2 of pplox which supports all tokens in the Lox language.

With the completion of the pplox tokenizer, we wanted to host it online and made a simple Django web application which runs pplox on the server.
